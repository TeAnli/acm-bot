# 专为西南科技大学 SCPC 竞赛平台 打造的 ncatbot 机器人插件

## 功能简介

这是一个专为西南科技大学 SCPC 竞赛平台设计的机器人插件，集成多种实用功能，方便用户快速获取相关竞赛信息。

### 主要功能

- **用户信息查询** - 一键获取 SCPC 平台用户基本信息
- **Codeforces 信息** - 查询用户的 Codeforces 比赛数据和评级
- **比赛记录查询** - 查看用户的比赛参与历史和成绩
- **排名系统** - 获取当前平台用户排名情况
- **男神图片** - 随机发送一张男神的图片

- **快速响应** - 简洁高效的命令交互体验
## 安装使用

### 环境要求

- Python 3.7+
- ncatbot 框架
- 具体依赖查看requirements.txt
